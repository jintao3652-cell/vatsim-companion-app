## 常见问题

### ❌ 权限对话框未弹出
**解决方案:**
1. 卸载应用
2. 清除应用数据
3. 重新安装运行

### ❌ 通知未显示
**Android:**
- 检查: 设置 > 应用 > VATSIM Companion > 通知 > 确认已开启
- 检查: 设置 > 通知 > 确认未开启勿扰模式

**iOS:**
- 检查: 设置 > 通知 > VATSIM Companion > 确认已开启
- 检查: 使用真机测试（模拟器可能不显示横幅）

### ❌ 编译错误
**解决方案:**
```bash
flutter clean
flutter pub get
flutter run
```

## 发布前清理

### 移除测试代码
在两个项目的 `lib/main.dart` 中删除:

```dart
// 删除这段测试代码
Future.delayed(const Duration(seconds: 2), () {
  _pushService.showTestNotification();
});
```

### 构建发布版本
```bash
# Android
flutter build apk --release
flutter build appbundle --release

# iOS
flutter build ios --release
```
